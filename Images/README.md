# Exploratory Data Analysis (EDA) – Image Collection

This folder contains the exploratory data analysis (EDA) visuals generated during the initial phase of the Home Credit Default Risk project. These images supported my understanding of:

- The structure of the dataset  
- Distributions of key demographic and financial variables  
- Target imbalance (default vs. repaid)  
- Missing value patterns across features  
- Relationships between predictors  
- Correlations among numeric variables  
- Early signals related to borrower behavior and risk  

The EDA visuals helped identify important issues such as skewed distributions, high-missing variables, non-linear patterns, and feature interactions. These insights directly informed the preprocessing steps, feature engineering decisions, and the choice of modeling techniques used later in the project.
